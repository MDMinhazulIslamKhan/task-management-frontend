## Used Technology

- TypeScript
- NextJs
- Redux
  - Redux Toolkit
- Axios
- React Hook form
  - Hook Form Resolvers
- Yup
- Tailwind
- DaisyUI
- ESLint

## Main Functionality

- Registration and login
- Edit profile
- Change password
- View profile
- Create task
- Assign task
- Notification
- See all task with pagination and filtering
- View task details
- Update task ( only for task creator )
- Delete task ( only for task creator )
- Accept assigned task
- Cancel assigned task
- Complete task
- Give feedback on task
- Delete own feedback on task
